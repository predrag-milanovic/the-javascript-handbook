# Loops

JavaScript has several kinds of loops for running code repeatedly: classic `for` loops, `while` loops, and variations like `break`, `continue`, and `for...in`.

This chapter walks through the most common loop patterns and the control flow tools you’ll see in real-world code.

## Lessons in this chapter

- [01 – Loops](./01-loops.md) – the classic `for` loop and how its three parts (initialization, condition, final expression) work together.
- [02 – Break](./02-break.md) – exiting a loop early with `break`, including intentional “infinite” loops you control manually.
- [03 – Continue](./03-continue.md) – skipping the rest of the current iteration and jumping straight to the next one.
- [04 – While](./04-while.md) – looping while a condition stays true, useful when you don’t know the number of iterations in advance.
- [05 – For...in](./05-for..in.md) – iterating over an object’s enumerable string keys, and what to know about key order.
