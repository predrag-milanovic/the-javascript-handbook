# Arrays

This chapter introduces JavaScript arrays and a handful of core tools you will use with them every day: checking their size, copying and combining them, searching inside them, looping over their values, taking slices, working with `const`, and destructuring.

Use these sections as a guided tour or as quick reference when you need to recall a specific feature:

- [01 - Arrays](docs/08-arrays/01-arrays.md) – What arrays are, how to create them, access items by index, and add elements with `push`.
- [docs/08-arrays/02-array-lenght.md](docs/08-arrays/02-array-lenght.md) – The `length` property and how it tracks the number of items in an array.
- [docs/08-arrays/03-array-spread.md](docs/08-arrays/03-array-spread.md) – Using spread syntax (`...`) to copy arrays and build new ones from existing arrays.
- [docs/08-arrays/04-includes.md](docs/08-arrays/04-includes.md) – Checking whether an array (or string) contains a given value with `.includes()`.
- [docs/08-arrays/05-for..of-loops.md](docs/08-arrays/05-for..of-loops.md) – Looping over array values with `for...of`, and how it differs from `for...in`.
- [docs/08-arrays/06-slicing-arrays.md](docs/08-arrays/06-slicing-arrays.md) – Taking non‑destructive slices of arrays with `.slice()`, including negative indexes.
- [docs/08-arrays/07-const-arrays.md](docs/08-arrays/07-const-arrays.md) – How `const` works with arrays: reassignment vs. mutating contents.
- [docs/08-arrays/08-destructure.md](docs/08-arrays/08-destructure.md) – Destructuring arrays into variables, using the rest operator, and what happens when you over‑destructure.
