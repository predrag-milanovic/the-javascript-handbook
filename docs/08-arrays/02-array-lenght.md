# Array Length

Every array in JavaScript has a built-in [**`length`**](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/length) property that tells you how many items it currently contains.

```js
const foods = ["burger", "fries", "pizza"]

console.log(foods.length)
// Prints: 3
```

The `length` value updates automatically as you add or remove elements from the array.
