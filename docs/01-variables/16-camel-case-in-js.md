### Camel case in JavaScript

In some languages like Python, Ruby, and Rust, the common convention for variable names is **snake_case**.

In JavaScript, the most popular convention (and the one we use in this handbook) is **camelCase**.

Here are some common casing styles you might see:

- `camelCase`
- `snake_case`
- `PascalCase`
- `SCREAMING_SNAKE_CASE`

When writing JavaScript code, prefer **camelCase** for variable and function names.
