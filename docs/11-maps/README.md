# Maps

Maps are JavaScript’s built-in key/value collection type. This chapter covers how to create them, how their keys behave, and when they are a better choice than plain objects.

## Lessons

1. [01 - Maps](./01-maps.md) introduces map basics, including setting, deleting, and copying entries.
2. [02 - Map Keys](./02-map-keys.md) explains what kinds of values can be used as keys and why reference equality matters.
3. [03 - Map vs. Object](./03-map-vs.-object.md) compares maps with plain objects and outlines the practical tradeoffs.
