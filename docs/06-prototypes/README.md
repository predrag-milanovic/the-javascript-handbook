# Prototypes

How JavaScript uses prototypes and prototype chains as its real inheritance system under the hood, powering objects, `class` syntax, and property lookup across linked objects.

## Lessons in this chapter

- [01 – Prototypal inheritance](./01-prototypal-inheritance.md) – creating objects that inherit from other objects, how `Object.create` works, and how child objects can shadow properties on their prototypes.
- [02 – Prototype chains](./02-prototype-chains.md) – how objects link together into chains, how property access walks those chains, and how `Object.getPrototypeOf` reveals what an object inherits from.
